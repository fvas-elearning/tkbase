Detailed Description

Use this iterator to create an options list from an array.

<?php $iterator = new ArrayArrayIterator(array( array('– Select –', ''), array('Admin', 'admin', true, 'label'), array('Moderator', 'moderator', false, 'label'), array('User', 'user')) ); ?>

Each sub array should contain the following structure:

array ( 0 => 'Text', // Option Text 1 => 'Value', // Option value (optional) 2 => false, // Option Disabled value (optional) 3 => 'label' // Option Label (optional) )
