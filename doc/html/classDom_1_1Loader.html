Detailed Description

Class Loader Use this class to facilitate with the loading of template files

You can add loader adapters to find templates in a cascading array

NOTE: Adapters are run as as LIFO (Last In First Out) queue. Michael Mifsud info@.nosp@m.trop.nosp@m.otek..nosp@m.com http://www.tropotek.com/ Copyright 2015 Michael Mifsud
